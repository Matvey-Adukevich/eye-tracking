# eye-tracking - Трекинг взгляда на основе нейросетей

## 🎯 Цель проекта

Проект представляет собой систему для определения положения зрачка на изображениях глаз с использованием нейронных сетей. 

**Основные задачи:**
- Обучение нейронной сети для предсказания координат зрачка
- Обработка и подготовка датасетов изображений глаз из MPIIGaze и Helen Eye Dataset
- Визуализация результатов работы модели и анализ ошибок предсказания
- Предсказание точки, куда смотрит пользователь

**Принцип работы:**
Модель принимает на вход изображение с вебкамеры и возвращает нормализованные координаты центра зрачка (от 0 до 1) для каждого глаза и точку на экране, куда смотрит человек.



## Установка и запуск

```bash
cd run
pip install -r requirements.txt
python main.py
```
